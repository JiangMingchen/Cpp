------------------------欢迎使用存款利息计算系统------------------------

使用本程序前请仔细阅读本说明，并按照正确的操作方法进行操作！

1.进入程序后进入登陆界面，按照操作提示进行注册和登录。（首次进行操作请先注册）
2.显示登录成功后按回车键后可进行三种操作，按操作提示可进行利息计算，用户计算记录查询，利率表查看三种操作。

~~~~利息计算~~~~

本系统利息计算方式如下：

（1）活期利息计算：
?利息按分起息，结算利息时按分结算，不足分的四舍五入。
?一个月按30天计算。
例如：1.14――2.18 = 34天
         2.18――3.16 = 28天
?计算复息。
?利息按季度结算，不足一季度直接结算，超过一季度结算完毕后归入本金
?每个季度最后一个月的21号（3.21，6.21，9.21，12.21）结算利息，结算后利息归入本金继续进行计算。
?按照结息当日的利率计算。

（2）定期利息计算：
?定期存款种类分为：三个月、六个月、一年、两年、三年、五年。最短为三个月，最长为五年。
?按照存款当日该种类的利率进行计算。
?无复息。

（3）约转：
?只在定期存款中存在
?当定期存款到期后，用户没有按时取出，如果用户在存款时选择进行约转，则在提款日自动按照同样的定期存款种类进行续存。

~~~~用户计算记录查询~~~~

每当用户进行一次计算操作后，数据会保存到用户计算记录中，需要时可以按操作提示进行查看。

~~~~利率表查看~~~~

（1）按操作提示进行利率表查看，利率表保存在rate.txt中，每行分别为：

存款年 存款月 存款日 活期利率 三个月利率 六个月利率 一年利率 两年利率 三年利率 五年利率

（2）本系统自带利率表为中国银行2007~2011年的利率信息，如果有其他需求请自行按照格式修改rate.txt文档。
        特别注意：请按照由早到晚的时间顺序进行利率信息的排列，否则可能会出现计算出错的情况。


注：本系统需定期维护。请勿删除任何信息文件（如data.txt web1.txt等）
